# 张京杰的个人主页

### 学习与项目

#### **短期目标**
* **HVAC 实验复现**：通过机器学习方法，复现一个与暖通空调（HVAC）相关的实验，探索模型在不同数据集上的泛化能力。

#### **长期目标**
* **单片机进阶**：系统学习不同型号的单片机架构和底层开发，掌握更复杂的嵌入式系统设计。
* **无人机项目**：计划未来将嵌入式知识与无人机技术相结合，探索飞行控制和传感器集成应用。
